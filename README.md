# MY-FIRST-PPT 🎉

A beautiful presentation built with [Reveal.JS](https://revealjs.com/) - Made with 💖

## 📋 Description

This repository contains a Reveal.JS-based presentation framework. Reveal.JS is a powerful tool for creating stunning HTML-based slideshows that work seamlessly across all modern browsers.

## 🌟 Features

- **HTML-based slides** - Write your presentations in clean, semantic HTML
- **Reveal.JS framework** - Professional presentation capabilities
- **Responsive design** - Works beautifully on all screen sizes
- **Easy customization** - Modify themes, layouts, and styling to match your needs

## 🛠️ Technologies Used

- **HTML** - Core presentation markup (100%)
- **Reveal.JS** - Powerful presentation framework

## 📖 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Git (for cloning the repository)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/hex32-iitkgp/MY-FIRST-PPT.git
   cd MY-FIRST-PPT
   ```

2. Open the presentation:
   ```bash
   # Simply open the HTML file in your browser
   open index.html
   # or navigate to the file in your file explorer and double-click it
   ```

## 🎨 Customization

- Edit the HTML files to modify slide content
- Customize themes and styling in the CSS files
- Refer to [Reveal.JS documentation](https://revealjs.com/) for advanced features
